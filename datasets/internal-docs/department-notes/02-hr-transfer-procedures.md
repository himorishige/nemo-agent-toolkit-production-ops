# 人事部 異動手続きノート

社員の異動（部署変更、昇進、降職）に伴う手続きのチェックリストです。**社員氏名・社員 ID を含む** ため人事部内および当該社員上長のみ参照可。

## 直近の異動例（架空）

| 社員 ID    | 氏名       | 異動前         | 異動後         | 発令日     |
| ---------- | ---------- | -------------- | -------------- | ---------- |
| example-0123 | 山田 太郎  | 営業部         | 営業企画部     | 2026-04-01 |
| example-0234 | 佐藤 花子  | カスタマーサポート部 | 営業部       | 2026-04-01 |
| example-0345 | 鈴木 一郎  | 開発部         | プロダクト企画部 | 2026-03-01 |

## 異動の手続きチェックリスト

### 異動の 1 か月前

- 本人面談（人事部）
- 受入れ部署との顔合わせ調整
- 引き継ぎ計画の作成（現所属上長）

### 異動の 2 週間前

- 引き継ぎドキュメントのレビュー
- アカウント権限の変更申請（情シス部）
- 業務関連 SaaS のグループ移動

### 異動当日

- 人事部での発令確認
- 受入れ部署でのオリエンテーション
- 旧 / 新部署への通知メール

### 異動の 1 か月後

- 適応状況のフォローアップ面談（本人 + 受入れ上長 + 人事担当）
- 必要に応じて追加サポート

## 担当連絡先

異動に関する相談は人事部の `hr@example.com`、または各社員の人事担当（社員ごとに割り当て）まで。
