# 業務デバイス管理ポリシー

会社貸与のノート PC・スマートフォン・外部記憶媒体に関するポリシーです。

## 貸与デバイス

- ノート PC: 入社時に貸与、原則 4 年間使用
- スマートフォン: 業務上必要な場合に申請ベースで貸与
- 外部記憶媒体（USB、外付け SSD）: 個別申請、用途終了後に返却

## セキュリティ要件

- ディスク暗号化必須（macOS は FileVault、Windows は BitLocker）
- スクリーンロック自動起動 5 分以内
- OS / ブラウザ / 業務アプリは月次でアップデート
- 業務外アプリのインストールは情シス承認が必要

## 紛失・盗難

- 紛失・盗難の発覚時は **即時** 情シス部のインシデント窓口（`it-incident@example.com`）に連絡
- 24 時間以内に紛失届を提出
- 端末のリモートワイプを情シス部が実施

## 退職時

- 退職日の 3 営業日前までに全機器を情シス部窓口に返却
- データの持ち出しは禁止（退職後も含む）

## 私物デバイスの業務利用

- 私物 PC・スマートフォンでの業務利用は原則禁止
- 例外承認はチーム長 + 情シス部長の連名稟議が必要
