# 在宅勤務時の通信ガイドライン

在宅勤務（リモートワーク）における通信環境の最低基準です。

## ネットワーク要件

- 自宅 Wi-Fi は WPA3 / WPA2 のいずれか（WEP 禁止）
- ルーターのファームウェアは最新版
- ゲストネットワークと業務 Wi-Fi の分離を推奨

## VPN

- 社内システムへのアクセスは VPN（Cloudflare WARP）経由必須
- 公共 Wi-Fi での業務時は必ず VPN を有効化
- VPN 設定の更新は社内ポータルから入手

## カフェ・コワーキングスペース

- 機密情報を含む画面の作業は不可（公共空間でのスクリーン覗き見対策）
- 通話・ビデオ会議はヘッドセット必須

## トラブルシュート

- VPN が繋がらない: 社内ポータル「VPN ヘルプ」を参照
- 解決しない場合: 情シス部 ヘルプデスク（`it-helpdesk@asahi-systems.example.com`）

セキュリティに不安がある場合は、無理せず情シス部に相談してください。
